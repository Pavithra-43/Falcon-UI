Gatling

Performance/Load Testing Tool

Why Gatling?

To integrate on top of a codebase for performance testing to compare performance in between multiple environments, or it can be used to compare performance of code written in multiple languages/framework.


How it works?

Gatling is written in Scala, the most emerging advanced language which allows to run it on any system, different local machines and cloud servers to run and create tests.

Gatling offers detailed metrics dashboard to see after tests execution without having to add any additional plugins.

The report is stored as an HTML file, which can be easily saved for future analyses and metrics comparison.

Report is interactive. which allows you to perform more detailed analyses and concentrate specific requests in addition to the overall picture.


Run the pipeline to see performance report!
Advantages

1. Avoid crashing: Gatling helps to anticipate slow response times and crashes.

2. Improve time-to-market: Gatling detects performances issues and errors.

3. Enhance user experience: Gatling gives an accurate picture of slowest user experience.

4. Ability to create completely scripted tests without a UI
