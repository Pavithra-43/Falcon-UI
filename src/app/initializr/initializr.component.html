Why Custom Initializer?

Custom Initializer is API based tool to easily generate the project structure of the Spring Boot Project. It offers extensible API for creating JVM-based projects. It also provides various options for the project that are expressed in a metadata model.


How to use custom initializer?

It is a default code generator that creates template with the required folder structure and dependencies with design patterns. Repeated integration jars can be included as and when required.

Follow below steps to create your application using custom initializer!

1. Go to custom initializer page and add your plugins, dependencies, packages and back-end details.

2. If you add packages, it will automatically create classes along with class level annotations.

3. Click download button to download your application.

4. You can download the project in zip format in your local machine.

5. Extract the file and import the project into IDE and proceed.


Click the button to create your application using custom-initializr



Advantages

1. Reduces manual work of building the pom file dependencies.

2. Repetitive dependencies and jars can be added to the project avoiding the manual addition of dependencies. Uniformity of the project structure can be followed for multiple apps.
