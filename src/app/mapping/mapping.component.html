Mapping Document

Reduce time in specification phase!

Why Mapping Generator?

1. Mapping generator is used in migration.(Tibco to any other open source migration)

2. Tibco code needs to be converted into specification document which can be understand by developer, which will be input to developer.

3. As tibco is being proprietary product, there are no tools available in the market which can convert the code into document


How Mapping Generator Works?

1. Mapping Generator tool interprets tibco code and generate a mapping sheet which helps to generate specification document.

2. This tool can interpret validation and transformation components of a tibco service.

3. From Tibco Studio, copy xslt of mapper activity and save it as xml File. The Python script will be executed by taking the XML file as an argument and the specification document will be generated as output.

Advantage of Mapping Generator

1. Reduces the effort of preparing specification document.

2. Minimizes the human error during specification phase, slippages and cost of re-work gets minimized.
